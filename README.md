# Gestión de Vehículos POO

Aplicación de consola en **C#** para gestionar vehículos usando principios de **POO**: atributos, métodos, parámetros y encapsulamiento.

## Herramientas utilizadas
- **Clases y objetos**: Vehiculo y Program.
- **Encapsulamiento**: atributos privados con propiedades get y set.
- **Validaciones**: año válido, precio mayor a 0, texto no vacío.
- **Colecciones**: List<Vehiculo> para almacenar vehículos.
- **Interacción**: menú en consola con Console.ReadLine() y Console.WriteLine().
- **Formato de moneda**: precio.ToString("C0", new CultureInfo("es-MX")).
